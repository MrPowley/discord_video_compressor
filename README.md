# [FR] Compresseur vidéo discord


Ce programme en python sert a compresser des vidéos/clips pour qu'il fassent moins de 25Mo pour pouvoir les envoyer sur discord

Il utilise les librairies suivantes:

- **subprocess**
- **pymediainfo**
- **datetime**
- **os**
